⚡ Superpowers

Some features need a little more

The Figma plugin and Chrome extension work great on their own. But certain features unlock only when you connect to Claude via the local MCP server. We call these Superpowers - Claude acting as the brain while the plugin and extension act as the hands.

🔗
What are Superpowers?
Superpowers is the connection between Claude and NiCE Designer's tools. Start the local MCP server, connect Claude Desktop, and Claude can drive both Figma and Chrome simultaneously using natural language - running multi-step workflows that neither tool could do alone.
🎯
What requires Superpowers?
Any workflow that spans both clients: Figma-to-Chrome parity checks, pushing an artboard's styles to a live page, AI-driven multi-step conversions, and all 9 Claude skills. Single-client audit and convert features work standalone.
🚀
Getting started
Install the MCP server from the download package, run node src/index.js in the packages/mcp-server/ folder, and connect Claude Desktop. See the Install Guide for the full setup walkthrough.
