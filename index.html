What it is

Three tools, one workflow

NiCE Designer brings design system compliance into Figma, the browser, and your AI assistant.

🎨

Figma Plugin

Audit and convert Figma frames - colors, type, spacing, radius, components, accessibility. Live compliance scores and token-level fixes.

🌐

Chrome Extension

Inspect, re-theme, and audit live web pages in real time. Edit mode lets you apply design tokens directly to the browser DOM.

🤖

MCP Server

WebSocket hub on port 7331 connecting Claude to both clients. 35 MCP tools and 9 Claude skills for AI-driven auditing and conversion.

FAQ

Common questions

How do the NiCE Designer tools understand the design system?

The tools are built on three sources of truth:

  • Storybook - the design team's live component library at na1.dev.nice-incontact.com/sol, which provides live component definitions, variants, and usage examples.
  • Figma foundation files - the Lyra and SOL Foundation Figma files, which define the canonical token sets, spacing scales, color palettes, and typography styles for both systems.
  • Claude design system knowledge - AI context built from the same sources, enabling Claude to reason about tokens, patterns, and compliance in natural language.
Does NiCE Designer generate designs or pages?

No. NiCE Designer works with pages and designs that already exist - it audits, converts, and annotates them rather than generating new ones from scratch.

The philosophy: generation is best left to tools like Claude Code for code and to UX designers for design. NiCE Designer's job is to ensure that what's already been built actually matches the design system.

What are Superpowers?

Superpowers is what happens when NiCE Designer connects to Claude via the local MCP server. On their own, the Figma plugin and Chrome extension can audit and convert designs. With Superpowers connected, Claude can drive both tools simultaneously using natural language - running multi-step workflows, reasoning across Figma and the live page at the same time, and using the full library of MCP tools and Claude skills.

Think of the plugin and extension as the hands, and Superpowers as the brain.
